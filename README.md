# nonz250's storage

This project is your own storage server.

## Project setup

```shell
make setup
```

## Start app server

```shell
make up
```

## Stop app server

```shell
make down
```

## Project testing for development

```shell
make test
```

## Project testing for production

Please execute this command before pull request.

```shell
make prod
```

## Execute code fixer

```shell
make fix
```
